A web-based ATS (Applicant Tracking System) Resume Analyzer built using Flask + LLM API integration.

This tool analyzes a resume against a job description and generates:

✅ ATS Score (/100)

✅ Structured Professional Evaluation

✅ Strengths & Weaknesses

✅ Actionable Improvement Roadmap

✅ Downloadable PDF Report
Features

Web UI built with HTML + Flask

AI-powered resume evaluation

Structured output (not messy paragraphs)

Clean UI with styled results section

Automatic PDF generation of analysis

Generic — works for any field (engineering, tech, business, etc.)

Tech Stack

Python

Flask

HTML/CSS

ReportLab (PDF generation)

Hugging Face / LLM API
